### Hardware

I already has a Rasberry PI at home, and needed to get just one more machine with enough resources to run a Proxmox. I could have gone with another Raspberry Pi but this wouldn't have let me install a hypervisor on it and run VMs comfortably.

```
1 - Rasberry Pi 4
1 - Dell Optiplex
```

### OS

I knew I wanted to run Ubuntu across all nodes, as this open-source freely available Debian based Linux OS available.

```
Raspberry Pi - Ubuntu
Dell Optiplex - Proxmox
```

### Infrastructure Automation

Terraform to manage everything, as I already had familiariaty.

Proxmox has a few available providers.

```
VM Provisioning - Terraform
OS Package Management - Ansible
```

### Kubernetes Flavour

```
kubeadm
```
